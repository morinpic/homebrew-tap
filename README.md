# Homebrew Tap

Custom Homebrew tap by [morinpic](https://github.com/morinpic).

## Install

```bash
brew tap morinpic/tap
```

## Available Casks

### Claude Status Bar

> Never miss a Claude outage.

macOS menu bar app that monitors Claude service status and sends desktop notifications on incidents.

```bash
brew install --cask claude-status-bar
```

Requires macOS 14.0 (Sonoma) or later.
